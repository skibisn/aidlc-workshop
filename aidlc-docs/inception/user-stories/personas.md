# 페르소나 정의

## Persona 1: 디지털 친숙 고객 (Digital-Savvy Customer)

- **이름**: 김민준
- **나이**: 20~35세
- **특징**: 스마트폰과 앱 사용에 익숙하며, 빠르고 직관적인 UI를 선호
- **목표**: 직원 호출 없이 빠르게 원하는 메뉴를 주문하고 싶음
- **불편사항**: 메뉴판을 기다리거나 직원을 부르는 시간 낭비
- **기술 수준**: 높음
- **관련 Stories**: US-C-01, US-C-02, US-C-03, US-C-04, US-C-05

---

## Persona 2: 디지털 비친숙 고객 (Digital-Unfamiliar Customer)

- **이름**: 이순자
- **나이**: 50~70세
- **특징**: 디지털 기기 사용에 익숙하지 않으며, 큰 글씨와 단순한 UI를 선호
- **목표**: 복잡한 조작 없이 원하는 메뉴를 주문하고 싶음
- **불편사항**: 복잡한 화면 구성, 작은 버튼, 이해하기 어려운 용어
- **기술 수준**: 낮음
- **관련 Stories**: US-C-01, US-C-02, US-C-03, US-C-04, US-C-05

---

## Persona 3: 외국인 고객 (Foreign Customer)

- **이름**: James Wilson
- **나이**: 25~45세
- **특징**: 한국어를 읽기 어려우며, 이미지와 가격 숫자에 의존하여 메뉴를 선택
- **목표**: 언어 장벽 없이 원하는 메뉴를 주문하고 싶음
- **불편사항**: 한국어로만 된 메뉴명/설명, 직원에게 말로 설명하기 어려움
- **기술 수준**: 중간 (스마트폰 사용 익숙)
- **관련 Stories**: US-C-01, US-C-02, US-C-03, US-C-04, US-C-05

---

## Persona 4: 매장 직원 (Store Staff)

- **이름**: 최현우
- **나이**: 20~30세
- **특징**: 홀 서빙 담당으로 주문 상태 확인 및 고객 응대를 담당
- **목표**: 어떤 테이블에 어떤 주문이 들어왔는지 빠르게 파악하고 싶음
- **불편사항**: 주문 상태 변경을 위해 카운터까지 이동해야 하는 불편함
- **기술 수준**: 중간
- **관련 Stories**: US-A-02, US-A-03

---

## Persona 5: 매장 관리자 (Store Manager)

- **이름**: 박지영
- **나이**: 30~50세
- **특징**: 매장 운영 전반을 담당하며, 주문 현황을 실시간으로 파악해야 함
- **목표**: 들어오는 주문을 빠르게 확인하고, 테이블 상태를 효율적으로 관리하고 싶음
- **불편사항**: 주문 누락, 테이블 정산 오류, 메뉴 변경의 번거로움
- **기술 수준**: 중간
- **관련 Stories**: US-A-01, US-A-02, US-A-03, US-A-04
