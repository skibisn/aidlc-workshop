## Question 1
User Story 구성 방식을 어떻게 하시겠습니까?

A) Feature-Based - 시스템 기능(메뉴 조회, 주문 생성 등) 중심으로 구성
B) Persona-Based - 고객/관리자 사용자 유형별로 구성
C) User Journey-Based - 사용자 워크플로우 흐름 중심으로 구성
D) Epic-Based - 대분류 Epic 아래 세부 Story 계층 구조로 구성
E) Other (please describe after [Answer]: tag below)

[Answer]: C

## Question 2

A) 표준 형식: "As a [persona], I want [goal], so that [benefit]" + 인수 기준
